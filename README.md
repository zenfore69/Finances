Агро-Расходы Мобильное приложение "Агро-Расходы" предназначено для автоматизации подачи и согласования авансовых отчетов и корпоративных трат в агропромышленном комплексе. Оно реализует ролевую модель доступа и интегрируется с системой 1С:Предприятие для синхронизации данных. Цели приложения

Обеспечить мобильный инструмент для фиксации расходов сотрудниками. Автоматизировать процесс согласования трат руководителями. Реализовать роли: Администратор, Сотрудник, Согласующий. Предоставить статистику расходов и настройки профиля.

Функционал

Авторизация: Вход по табельному номеру и паролю. Создание расхода: Указание суммы, категории, фото чека; отправка на согласование. Просмотр истории: Список расходов с поиском, фильтрами и статусами (На проверке, Одобрено, Отклонено). Согласование: Для роли Согласующий — просмотр списка, одобрение/отклонение. Управление пользователями: Для Администратора — создание аккаунтов с назначением ролей. Статистика: Общие суммы, breakdown по категориям. Настройки: Профиль, синхронизация с 1C, выход из системы. Синхронизация: Автоматический обмен статусами с базой.

Прототип и интерфейс Прототип создан в draw.io. Основные экраны:

Экран авторизации: Поля "Табельный номер" и "Пароль". Главное меню: "+ Новый расход", "История отчетов", "Статистика". Оформление чека: Сумма, выбор категории (ТСМ, Канцелярия, Хоз., Прочие), фото чека. Мои расходы: Поиск, фильтры, список карточек. Статистика: Общие расходы и по категориям. Согласование: Детали расхода с кнопками "Одобрить" / "Отклонить". Пользователи: Форма создания с чекбоксами ролей. Настройки: Профиль, синхронизация, версия, "Выйти".

Руководство пользователя

Вход в систему: Введите табельный номер и пароль, нажмите "ВОЙТИ". Создание расхода: Нажмите "+ Новый расход", укажите сумму, категорию, сфотографируйте чек, нажмите "ОТПРАВИТЬ В 1C". Просмотр истории: В "Мои расходы" используйте поиск и фильтры; просмотрите статусы. Просмотр статистики: В "Статистика" увидите суммы по категориям. Согласование (для Согласующего): В "Согласование" выберите расход, нажмите "Одобрить" или "Отклонить". Управление пользователями (для Администратора): В "Пользователи" заполните форму, назначьте роли, нажмите "СОЗДАТЬ ПОЛЬЗОВАТЕЛЯ". Настройки: Просмотрите профиль, статус синхронизации, нажмите "Выйти".
